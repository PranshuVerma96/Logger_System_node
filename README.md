# Node Logger System

This is a simple logger system built using Node.js.

## 🚀 Features

* Save logs in a file
* Add date and time to each log
* Append data (no overwrite)
* Modular code structure

## 📚 Concepts Used

* Node.js modules (`require`, `module.exports`)
* fs module
* appendFileSync
* Date and time handling

## 🛠️ Technologies

* JavaScript (Node.js)

## ▶️ How to Run

```bash
node index.js
```

## 📄 Example Output (log.txt)

```
[20/03/2026, 5:00:00 pm] Server started
[20/03/2026, 5:00:01 pm] User logged in
[20/03/2026, 5:00:02 pm] Error occurred
```

## 🎯 Purpose

This project helps beginners understand how logging works in real-world backend applications. It stores important events and errors in a file for debugging and monitoring.

## 💡 Future Improvements

* Add log levels (INFO, ERROR)
* Create separate error log file
* Use async methods
